# V2Mate — YouTube & Instagram Video Downloader

A free, fast video downloader built with Next.js 15. Powered by Cobalt API.

## 🚀 Deploy to Vercel (FREE)

### Step 1 — Upload to GitHub
1. Go to github.com and sign in
2. Click the **+** button → **New repository**
3. Name it: `v2mate`
4. Keep it Public, click **Create repository**
5. Click **uploading an existing file**
6. Drag and drop this entire project folder
7. Click **Commit changes**

### Step 2 — Deploy on Vercel
1. Go to vercel.com and sign in with GitHub
2. Click **Add New → Project**
3. Find your `v2mate` repo and click **Import**
4. Leave all settings as default
5. Click **Deploy**
6. Wait ~2 minutes → Your site is live! 🎉

Your URL will be: `https://v2mate.vercel.app`

## 💻 Run Locally (for testing)

```bash
npm install
npm run dev
```

Then open: http://localhost:3000

## ✅ Features
- Download YouTube videos (360p → 1440p)
- Download YouTube Shorts
- Download Instagram Reels

- Mobile responsive
